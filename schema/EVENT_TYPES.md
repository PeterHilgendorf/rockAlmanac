# Event Types

The controlled vocabulary for `event.event_type`. Adding a new type is a
schema change: it gets added here first, with a definition, then used.

| Type | Definition |
|---|---|
| FORMATION | Artist is formed |
| NAME_CHANGE | Artist changes name |
| MEMBER_JOIN | Person joins artist (mirrors a membership start) |
| MEMBER_EXIT | Person leaves artist (mirrors a membership end) |
| MEMBER_RETURN | Person rejoins (mirrors a new membership record) |
| ALBUM_RELEASE | Album released (date derived from album record) |
| SINGLE_RELEASE | Single released |
| TOUR_START | Tour begins |
| TOUR_END | Tour ends |
| CONCERT | A single historically significant show |
| RECORDING_SESSION | Notable session begins/occurs |
| LABEL_CHANGE | Artist signs to / leaves a label |
| AWARD | Grammy, Hall of Fame induction, etc. |
| CHART_MILESTONE | e.g. album reaches #1 |
| SIDE_PROJECT | Member launches side project |
| HIATUS_START | Artist goes on hiatus |
| REUNION | Artist reunites |
| BREAKUP | Artist disbands |
| DEATH | Person dies |
| MARRIAGE | Person marries (person_ids lists both when both are in the dataset; pairs with a MARRIED_TO relationship). A remarriage is simply another MARRIAGE event |
| DIVORCE | Marriage ends (closes the MARRIED_TO relationship's end_date) |
| ARREST | Person arrested / drug bust / legal trouble — description carries the specifics |
| PERSONAL_EVENT | Notable personal moment that shaped the story: death of a child, health crisis, religious conversion, sobriety — description carries it |
| TV_APPEARANCE | Major television moment: network debut, Ed Sullivan, SNL, The Late Show |
| PRESS_MILESTONE | Major press moment: cover of Rolling Stone, landmark interview or profile |
| FILM | A film about the artist is released — a biopic, concert film, or documentary |
| INCIDENT | Editorially significant one-off that fits nothing above (accident, onstage brawl, controversy) |

Rules:
- Personal-life events (MARRIAGE, DIVORCE, ARREST, PERSONAL_EVENT) exist
  because they shaped the music and the story — Rock Almanac is a
  reference work, not a tabloid. The significance field and editorial
  judgment decide what's in.
- MARRIAGE/DIVORCE events and MARRIED_TO relationships are two views of
  one fact: the events mark the dated moments; the relationship carries
  the durable link. A verify script checks they agree.
- Membership-related events must agree with membership.yaml records.
  A verify script will check this.
- ALBUM_RELEASE / TOUR_START / TOUR_END events are GENERATED by build
  scripts from album and tour records — never hand-entered.
